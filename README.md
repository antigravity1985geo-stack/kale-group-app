## 🚀 Overview
Kale Group ERP is a sophisticated, all-in-one management platform designed for modern businesses. Built with a focus on visual excellence and technical precision, it automates everything from inventory management to complex fiscal reporting using the Estonian-model profit tax system popular in Georgia.

## ✨ Key Features

### 📊 Advanced Accounting & Reporting
- **Dynamic Financial Reports**: Real-time Balance Sheet, Profit & Loss, and Trial Balance with customizable date range filters.
- **Estonian Tax Model**: Automated calculation and declaration of profit tax (15%) on dividend distributions.
- **Payroll Automation**: One-click payroll processing with automated 20% income tax calculations and ledger entries.
- **Double-Entry Ledger**: Full traceability of all financial transactions with a robust journal entry system.
- **Data Export**: Professional CSV/Excel export for all financial reporting modules.

### 📦 Supply Chain & Inventory
- **Smart Waybill Mapping**: Seamless integration with RS.GE for incoming and outgoing waybills.
- **FIFO Costing**: Accurate inventory valuation using First-In-First-Out layers.
- **Returns Management**: Intelligent RMA module with automated accounting reversals.

### 🛠 Manufacturing
- **BOM Management**: Comprehensive Bill of Materials for furniture manufacturing.
- **Production Costs**: Automated mapping of raw material consumption to finished goods.

## 🛠 Tech Stack
- **Frontend**: React (Vite) + TypeScript + Tailwind CSS / Vanilla CSS
- **Backend**: Node.js (Vercel API) + Express
- **Database**: Supabase (PostgreSQL)
- **AI Integration**: Google Gemini AI for smart search and data analysis.

## ⚙️ Installation & Setup

### Prerequisites
- Node.js (v18+)
- Supabase Project

### Local Development
1. **Clone & Install**:
   ```bash
   git clone [repository-url]
   cd kale-group
   npm install
   ```

2. **Environment Variables**:
   Create a `.env` file in the root:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   SUPABASE_SERVICE_ROLE_KEY=your_key
   GEMINI_API_KEY=your_key
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
